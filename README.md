This is a Java-based Cab Booking Web Application built using:
- Java (JSP & Servlet)
- MySQL Database
- Apache Tomcat Server

## Features
- User can book cab with phone, pickup, drop location
- Driver dashboard to view all bookings
- Accept / Reject booking requests
- Pending bookings highlighted
- Auto-refresh dashboard

## Tech Stack
Java | JSP | Servlet | JDBC | MySQL | HTML | CSS
